## 地理编码接口

#### /geocode 地理编码 GET
| 名称 | 描述 | 类型 | 必选 | 默认值 | 允许值 | 
| :---: | :---: | :---: | :---: | :---: | :---: |
| address | 地点名称 | string | 是 | / | / |
| city | 城市名 | string | 是 | / | 中文名、英文名、地区编码 |

#### /geocode/reverse 逆地理编码 GET
| 名称 | 描述 | 类型 | 必选 | 默认值 | 允许值 | 
| :---: | :---: | :---: | :---: | :---: | :---: |
| x | 逆地理编码的X值 | double | 是 | / | -180.0 ~ 180.0 |
| y | 逆地理编码的Y值 | double | 是 | / | -90.0 ~ 90.0 |
